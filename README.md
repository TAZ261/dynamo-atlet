# Динамо Атлет — QR сторінка

Статична сторінка для Instagram та швидкого додавання контактів.

## Як запустити локально

Відкрийте `index.html` у браузері.

## Публікація на GitHub Pages

1. Створіть новий GitHub репозиторій.
2. Завантажте в нього вміст цієї папки.
3. Увімкніть GitHub Pages (Settings -> Pages -> Deploy from branch).
